# 🏦 PyBank ATM System

A robust and secure command-line interface (CLI) banking application built with Python. This project demonstrates core programming concepts, including authentication, transaction management, and error handling.

## 📂 Project Structure
This repository contains two versions of the ATM System to demonstrate iterative development:

- `ATM-System-v1.py`: **Version 1.0** — The initial implementation focusing on basic banking functionality.
- `ATM-System-v2.py`: **Version 2.0** — A refactored version with enhanced code structure, input validation, and modern Python syntax.

## 🚀 Features
* **Secure Authentication**: PIN-protected login system with a 3-attempt security limit.
* **Transaction Management**:
  - Secure balance inquiry.
  - Deposit functionality with input validation.
  - Withdrawal system with real-time balance validation.
* **Robust Error Handling**: Utilizes `try-except` blocks to manage invalid user inputs gracefully.
* **Clean Menu Interface**: Uses Python’s `match-case` logic for a smooth user experience.
