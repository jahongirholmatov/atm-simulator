# ATM Simulator

A simple command-line interface (CLI) banking application written in Python. It allows users to perform basic transactions after authenticating with a PIN code.

## Features
- **Secure Authentication**: PIN-protected login with a 3-attempt limit.
- **Transaction Management**: 
  - Check account balance.
  - Deposit funds.
  - Withdraw funds (with balance validation).
- **User-Friendly Menu**: Continuous loop allowing multiple operations without restarting the program.

## How to Run
1. Ensure [Python](https://www.python.org/) is installed.
2. Download the `main.py` file.
3. Open your terminal or command prompt.
4. Run the script:
   ```bash
   python main.py
